# Веб приложение электронный документооборот
## О проекте
## Задание: Приложение должно реализовывать основные CRUD-операции:
Проект представляет собой веб-приложение, представляющее собой упрощенную систему электронного документооборота
## Используемые технологии:
- Spring Boot, 
- Spring Data JPA, 
- PostreSQL, 
- Maven
## Представляет собой веб-приложение с структурой:
-организация;
-функция;
-сотрудники;
-документы
